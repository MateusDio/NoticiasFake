📰 Portal de Fake News

Um sistema interativo onde notícias não são apenas lidas… elas são julgadas ⚖️ pela comunidade.

Cada publicação entra em um “tribunal digital”, e os usuários decidem se aquilo é confiável ou pura invenção.

⚙️ Como o sistema funciona
🧾 Publicação de notícias

Usuários autenticados podem criar notícias com:

Título
Resumo
Conteúdo completo

Assim que publicada, a notícia aparece imediatamente no feed.

🧑‍🤝‍🧑 Interação dos usuários

Cada notícia vira um campo de disputa de opiniões:

⬆️ Voto positivo → usuário considera confiável
⬇️ Voto negativo → usuário considera fake

O sistema calcula automaticamente:

Pontuação total
Quantidade de votos positivos
Quantidade de votos negativos
Voto individual do usuário
🧠 Lógica dos votos

O comportamento segue o estilo de redes sociais:

Cada usuário pode votar uma única vez por notícia
É possível mudar o voto
A pontuação pode ser positiva, negativa ou neutra

O número negativo não é erro, é a comunidade basicamente dizendo:
“isso aqui tá estranho…” 🕵️

🔍 Busca e filtros

O sistema permite:

Buscar notícias por palavras-chave
Filtrar conteúdos
Navegar pelo feed de forma dinâmica
📄 Visualização da notícia

Ao acessar uma notícia:

Exibe título, autor e data
Mostra o conteúdo completo
Mantém formatação legível
🧑‍💻 Dashboard do usuário

Cada usuário possui um painel onde pode:

Criar notícias
Editar suas publicações
Excluir conteúdos
Acompanhar desempenho (votos recebidos)

Tudo isso via modais, sem redirecionamento.

⚡ Sistema de modais

O sistema utiliza janelas sobrepostas:

Formulários abrem sem sair da página
A navegação fica mais rápida
Experiência semelhante a aplicações modernas
